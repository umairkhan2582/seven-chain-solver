# Contributing to Seven Chain Bridge Solver

  Thank you for contributing! 🌉

  ## Setup
  ```bash
  git clone https://github.com/YOUR_USERNAME/seven-chain-solver
  cd seven-chain-solver && npm install && cp .env.example .env
  git checkout -b feature/your-feature-name
  ```

  ## Priorities
  - 🚀 Faster fill strategies and prioritization
  - ⛓️ Additional chain support (Arbitrum, Base, Solana)
  - 📊 Local analytics dashboard
  - 🔒 Security improvements

  ## Standards
  - TypeScript — all code must be strictly typed
  - No console.log — use the built-in logger
  - Always handle async errors

  ## Submitting
  1. `npm run build` must compile cleanly
  2. Open a PR with clear title and description
  3. Reference issues with `Closes #123`
  4. PRs reviewed within **48 hours**
